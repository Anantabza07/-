📜 กติกา

  • 1นักผจญภัย 5 คน เริ่มที่ปากถ้ำ (ช่อง 0)
  • 2ตอบโจทย์ถูก = ได้เดินทันที ไม่ต้องรอคนอื่น
  • 3ตอบผิด = หยุดรอบนั้น ตอบซ้ำไม่ได้
  • 4ทุก 2 นาที มีปริศนาถ้ำ — ตอบถูกรอด ตอบผิดลาวาไล่ทัน!

จอยห้อง

ใส่รหัสห้อง 4 หลักที่เพื่อนแชร์มา

⚙️ วิธีตั้งค่า Firebase (คลิกเพื่อดู)
  1. เข้า console.firebase.google.com → สร้างโปรเจกต์ใหม่
  2. ไปที่ Build → Realtime Database → สร้าง database (เลือก test mode)
  3. ไปที่ Project Settings ⚙️ → Your apps → เพิ่ม Web app
  4. Copy firebaseConfig object ที่ได้
  5. เปิดไฟล์ firebase-config.js แล้วแทนที่ค่าตามที่ได้มา
// firebase-config.js
const FIREBASE_CONFIG = {
  apiKey: "YOUR_API_KEY",
  authDomain: "your-app.firebaseapp.com",
  databaseURL: "https://your-app-default-rtdb.firebaseio.com",
  projectId: "your-app",
  storageBucket: "your-app.appspot.com",
  messagingSenderId: "000000000",
  appId: "1:000:web:000"
};
